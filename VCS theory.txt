Git/GitHub
Git != GitHub
Git - это система контроля версий;
GitHub - это облачное хранилище;

Repository - это хранилище исходных кодов.
Local repository;
Remote repository - GitHub;

fetch - скачивает но НЕ распаковывает изменения с удаленного репозитория на локальный;
pull  - скачивает и распаковывает изменения с удаленного репозитория на локальный;
push  - отправляет изменения с локального репозитория на удаленный;

COMMIT - это контрольная точка, в которую сохраняются изменения;
Commit message - описывает изменения внесенные в репозиторий;
