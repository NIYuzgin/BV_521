Array - это набор переменных одного типа в НЕПРЕРЫВНОЙ ОБЛАСТИ ПАМЯТИ.

1. Динамические и Статические;
2. Одномерные, двумерные, трехмерные....... многмерные;


			type name[SIZE];
type - тип элементов массива;
name - имя массива. 
	   Для именования массивов используются такие же 
	   идентификаторы, как и для именования переменных.
SIZE - количество элементов массива, может быть задано лишь 
	   целочисленным константным значением;

	   const int n = 5;
	   int arr[n];
